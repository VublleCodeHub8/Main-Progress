### 🔄 Installation & Setup

```bash
# 1. Clone the repository
git clone https://github.com/VublleCodeHub8/Main-Progress.git
cd Main-Progress

# 2. Frontend setup
cd frontend
npm install
npm run dev    # 🌐 http://localhost:5173

# 3. Backend setup
cd ../backend
npm install
npm run start  # 🔌 http://localhost:3000
```
